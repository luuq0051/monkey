## [1.0.0] - 2026-04-20

### Added
- ✨ Initial release
- 🔌 Express.js server setup
- 📡 7 API endpoints (root, hello, cherry-studio, config, models, test/minimax, minimax/status)
- 🤖 Support for 8 AI providers (OpenAI, Anthropic, Google, Meta, Mistral, Cohere, Minimax, Local)
- 🔑 Environment variable configuration with dotenv
- 📖 Comprehensive documentation (README, API_DOCUMENTATION, INSTALLATION_GUIDE, DEPLOYMENT_GUIDE)
- 🤝 Contributing guidelines
- 📝 Changelog
- 🔒 Security guidelines

### Features
- Cherry Studio integration
- Minimax m2.7 model support
- RESTful API endpoints
- JSON responses
- Status checking endpoints
- Configuration validation

### Documentation
- README.md - Project overview
- API_DOCUMENTATION.md - Complete API reference
- INSTALLATION_GUIDE.md - Detailed setup instructions
- DEPLOYMENT_GUIDE.md - Deployment to multiple platforms
- CONTRIBUTING.md - Contribution guidelines
- SECURITY.md - Security best practices
- .env.example - Environment template

## Versioning Policy

Dự án này tuân theo **Semantic Versioning**:

- **MAJOR** (X.0.0) - Breaking changes
- **MINOR** (0.X.0) - New features, backward compatible
- **PATCH** (0.0.X) - Bug fixes, backward compatible

## How to Update

```bash
# Check for updates
npm outdated

# Update all packages
npm update

# Update specific package
npm update express
```
